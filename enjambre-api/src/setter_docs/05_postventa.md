# SERVICIO POST-VENTA

## OBJETIVO
Gestionar dudas e incidencias de clientes despues de contratar seguro, de forma rapida, clara y automatizable.
- Resolver dudas basicas sin intervencion humana
- Escalar solo casos importantes
- Reducir carga operativa

## TIPOS DE INCIDENCIAS PRINCIPALES

### 1. AHV / Tarjeta de seguro no llega
**Problema**: Cliente no recibe tarjeta del seguro.
**Causa habitual**: Falta numero AHV (seguridad social suiza). El cliente cree que lo genera el seguro, pero lo genera la empresa (empleador) o registro administrativo.
**Solucion**:
- Explicar que es el AHV
- Indicar que debe pedirlo a su empresa
- Confirmar que el seguro lo necesita para emitir la tarjeta

### 2. Facturas que no llegan
**Problema**: Cliente no recibe factura mensual.
**Proceso**: 
1. Cliente contacta
2. Se revisa
3. Se contacta con aseguradora (email interno)
4. Se solicita reenvio
**Automatizacion**: Bot informa que puede tardar X dias. Si pasa plazo -> escalar a humano.

### 3. Dudas sobre documentos (Ayuntamiento)
**Problema**: Cliente recibe papeles y no sabe cual enviar al ayuntamiento.
**Respuesta**: El documento es la "Versicherungspolice" (poliza). Es el documento que demuestra que tienes seguro.

### 4. Como pagar la factura
**Problema**: Cliente no sabe como pagar.
**Solucion**: Explicar QR suizo:
1. Abrir app del banco
2. Escanear QR
3. Confirmar pago

### 5. Cambios en el contrato (14 dias)
**Problema**: Cliente quiere modificar algo tras contratar (quitar complementarios, cambiar franquicia).
**Regla**: Dentro de 14 dias -> se puede modificar. Fuera de plazo -> NO.

### 6. Medico asignado
**Problema**: Cliente no sabe que medico tiene.
**Respuesta**: Esta en la poliza. Debe registrarse con ese medico.

### 7. Cambio de direccion
**Problema**: Cliente se muda.
**Proceso**: Cliente informa -> se notifica a aseguradora -> se actualizan datos.

### 8. Precio del seguro ("por que pago tanto?")
**Problema**: Cliente percibe precio alto.
**Respuesta**: Depende de edad, canton, franquicia, modelo.
**Objetivo**: Reforzar decision, explicar valor, evitar cancelacion.
